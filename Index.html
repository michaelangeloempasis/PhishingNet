Overview

PhishingNet provides a browser extension + backend to analyze pages for phishing risk. The extension captures URL and optional visible text and posts to /analyze. Backend returns JSON with score and phishing.

1) Capture

The extension reads the current tab URL (and, if enabled, visible text snippets) without storing personal data.

2) Analyze

Our backend evaluates lexical URL features and optional semantic signals to estimate a phishing risk score.

3) Decide

We display a clear verdict: SAFE or PHISHING, plus a score and short note to guide the user.

Why PhishingNet?

  • Real-time protection: checks run instantly while browsing.
  • Private by design: you choose what leaves the browser.
  • Actionable feedback: simple verdicts with supporting detail.
  • Lightweight: optimized heuristics with optional deeper analysis.
